# Flandre Card Pool Draft

## Core Direction

- フランドールの主軸は `破壊の目` を生成して壊すこと。
- デッキは次の3ループを回す想定。
  - `破壊の目` を置く
  - 全体攻撃、多段攻撃、専用札で `破壊の目` を壊す
  - 撃破誘発と `狂気` で連鎖ターンを伸ばす

## Current `破壊の目` Intent

- `破壊の目` は敵エリアに生成される特殊敵。
- HP は `max(5, ceil(対象の最大HP * 5%))`。
- `破壊の目` が失ったHPの一部をリンク先にも与える。
- `破壊の目` が壊れた時、リンク先へ爆発ダメージを与える。
- 同じ敵に `破壊の目` を重ねると、爆発ダメージが増える。
- パワー右側の数値は爆発ダメージを表す。

## Naming Rule

- 効果キーワード名は `破壊の目` で統一する。
- 初期カード名は `眼差し`。
- カード本文は短く保つ。
  - `破壊の目 1`
  - アップグレード後は `破壊の目 2`
- 詳細な挙動は `破壊の目` パワーのツールチップ側に寄せる。

## Starter Package

- 初期デッキ
  - `ストライク` x4
  - `防御` x4
  - `眼差し` x2
- 初期レリック
  - `破壊衝動`
  - `1ターンに1回、敵を倒した時、次のターン開始時に{Energy:energyIcons()}を得る。`

## `狂気` Specification

- 正式テキスト
  - `次のアタックは敵全体を対象にする。発動後、これを1失う。`
- 基本ルール
  - `1` スタックにつき、次のアタック `1` 回を全体化する
  - 発動したアタックの解決後に `狂気` を `1` 消費する
  - アタック以外のカードには影響しない
- 全体化の挙動
  - 単体攻撃はそのまま全体攻撃になる
  - 多段攻撃はヒット数を保ったまま全体化する
  - ランダム攻撃も全体化する
  - 例: `乱反射` は `3ダメージを3回、敵全体に与える` として扱う
- 設計意図
  - 単体高火力札を面制圧札へ変える
  - `破壊の目` をまとめて巻き込みやすくする
  - ボス戦でも雑魚戦でも腐らない爆発リソースにする

## Pool Structure Goal

- コモンは安定して `破壊の目` に触れる札を厚くする。
- アンコモンは `全体生成`、`連鎖生成`、`狂気` の伸びを担当する。
- レアは盤面全体を壊しきる爆発ターンの上限を担う。

## Common Cards

### 1. `眼差し`

- レア度: `コモン`
- コスト: `2`
- 種別: `スキル`
- 対象: `単体敵`
- 効果: `破壊の目 1`
- アップグレード: `破壊の目 2`
- 役割: 最も素直な単体設置札。初期デッキとテーマの基準。

### 2. `狂視`

- レア度: `コモン`
- コスト: `1`
- 種別: `スキル`
- 対象: `なし`
- 効果: `狂気` を1得る。ランダムな敵に `破壊の目 1`
- アップグレード: カードを1枚引く
- 役割: `狂気` の入口兼、軽い目の供給札。

### 3. `掃射`

- レア度: `コモン`
- コスト: `1`
- 種別: `アタック`
- 対象: `全体`
- 効果: 敵全体に `4` ダメージ
- アップグレード: 敵全体に `6` ダメージ
- 役割: もっとも基本的な起爆札。雑魚戦のテンポ札でもある。

### 4. `火花散らし`

- レア度: `コモン`
- コスト: `0`
- 種別: `アタック`
- 対象: `単体敵`
- 効果: 敵1体に `4` ダメージ。対象に対応する `破壊の目` があるなら、その `破壊の目` にも `4` ダメージ
- アップグレード: 敵と `破壊の目` へのダメージが `6` になる
- 役割: 目を狙って割りにいける軽量札。序盤の取り回し担当。

### 5. `裂き爪`

- レア度: `コモン`
- コスト: `1`
- 種別: `アタック`
- 対象: `単体敵`
- 効果: 敵1体に `7` ダメージ。対象に対応する `破壊の目` があるなら、追加で `4` ダメージ
- アップグレード: 基本ダメージ `10`、追加ダメージ `5`
- 役割: 目を置いた相手へ確実に圧をかける単体主力。

### 6. `乱反射`

- レア度: `コモン`
- コスト: `1`
- 種別: `アタック`
- 対象: `ランダム`
- 効果: ランダムな敵に `3` ダメージを `3` 回与える
- アップグレード: ヒット数が `4` 回になる
- 役割: `破壊の目` を拾いやすい多段札。筋力や `フォーオブアカインド` とも相性が良い。

### 7. `残骸あさり`

- レア度: `コモン`
- コスト: `1`
- 種別: `スキル`
- 対象: `なし`
- 効果: このターン、敵を倒すたびカードを `1` 枚引く
- アップグレード: コスト `0`
- 役割: `破壊の目` も含めた撃破連鎖の基本報酬札。

### 8. `ひび割れた笑み`

- レア度: `コモン`
- コスト: `1`
- 種別: `スキル`
- 対象: `なし`
- 効果: `6` ブロックを得る。場に `破壊の目` があるなら、追加で `4` ブロックを得る
- アップグレード: 基本ブロック `8`、追加ブロックは `4` のまま
- 役割: テーマに沿って使いやすい守り札。目を置くターンの隙を埋める。

## Common Package Notes

- コモン8枚のうち、`破壊の目` を新規生成できる札は `眼差し` と `狂視` の2枚。
- コモン段階では `全体生成` を入れず、単体設置から丁寧に回す形にする。
- `起爆札` は `掃射`、`火花散らし`、`乱反射` の3枚を軸にする。
- `撃破報酬` はまずドロー中心にして、エナジー回復は初期レリックに寄せる。
- `狂気` はコモンでは `狂視` のみとし、ばらまきすぎない。
- `乱反射` は `狂気` 時にランダムのまま散らず、敵全体へ同じヒット数を与える前提。

## Uncommon Cards

### 1. `視線増殖`

- レア度: `アンコモン`
- コスト: `2`
- 種別: `スキル`
- 対象: `全体`
- 効果: 敵全体に `破壊の目 1`
- アップグレード: コスト `1`
- 役割: 盤面全体へ一気に目を配る量産札。雑魚戦の連鎖起点。

### 2. `血の遊戯`

- レア度: `アンコモン`
- コスト: `1`
- 種別: `パワー`
- 対象: `なし`
- 効果: 敵を倒すたびカードを `1` 枚引く
- アップグレード: 戦闘開始時にカードを `1` 枚引く
- 役割: 目を含む撃破連鎖のエンジン。長いターンを成立させる。

### 3. `残響視`

- レア度: `アンコモン`
- コスト: `1`
- 種別: `スキル`
- 対象: `なし`
- 効果: このターン最初に敵を倒した時、ランダムな敵に `破壊の目 1`
- アップグレード: 発動回数が `このターン2回まで` になる
- 役割: 撃破から次の目へつなぐ連鎖生成札。

### 4. `ラーヴァテイン`

- レア度: `アンコモン`
- コスト: `2`
- 種別: `アタック`
- 対象: `全体`
- 効果: 敵全体に `10` ダメージ
- アップグレード: 敵全体に `14` ダメージ
- 役割: 目をまとめて削る中核の全体攻撃。ボス戦でも最低限腐らない基準火力。

### 5. `狂える瞳`

- レア度: `アンコモン`
- コスト: `1`
- 種別: `スキル`
- 対象: `なし`
- 効果: `狂気` を1得る。このターン、受けるダメージが `50%` 増加する
- アップグレード: カードを `1` 枚引く
- 役割: 高リスクで爆発ターンを作る `狂気` 供給札。

### 6. `過剰破壊`

- レア度: `アンコモン`
- コスト: `2`
- 種別: `アタック`
- 対象: `単体敵`
- 効果: 敵1体に `14` ダメージ。対象を倒したら、他の敵全体に `8` ダメージ
- アップグレード: 基本ダメージ `18`、波及ダメージ `10`
- 役割: 単体処理と面制圧を一枚で担う撃破報酬アタック。

## Uncommon Package Notes

- アンコモンでは `破壊の目` の供給を `単体` から `全体` と `連鎖` へ拡張する。
- `視線増殖` は雑魚戦用、`残響視` は連鎖継続用と役割を分ける。
- `血の遊戯` と `残骸あさり` は重なると強いが、どちらもドロー止まりでエナジーは増やしすぎない。
- `狂える瞳` はリスクを明確にして、ボス戦の爆発力へ寄せる。
- `ラーヴァテイン` と `過剰破壊` は、目に頼らなくても最低限使える打点札にしておく。
- `狂気` の本格運用はアンコモン以降に寄せ、コモン帯では入口だけに留める。

## Rare Cards

### 1. `フォーオブアカインド`

- レア度: `レア`
- コスト: `2`
- 種別: `パワー`
- 対象: `なし`
- 効果: この戦闘中、与えるダメージが `4` 倍になり、受けるダメージも `4` 倍になる
- アップグレード: コスト `1`
- 役割: フランドールの看板札。雑魚戦でもボス戦でも一気にゲームを傾ける。

### 2. `カタストロフビジョン`

- レア度: `レア`
- コスト: `2`
- 種別: `パワー`
- 対象: `なし`
- 効果: `破壊の目` が壊れるたび、他の敵全体に `6` ダメージ
- アップグレード: 追加ダメージ `8`
- 役割: 盤面全体を爆発に変える継続火力札。雑魚戦の制圧役。

### 3. `狂気解放`

- レア度: `レア`
- コスト: `1`
- 種別: `スキル`
- 対象: `なし`
- 効果: `狂気` を `2` 得る。このターン、次の攻撃のダメージを `+8` する
- アップグレード: 強化値 `+12`
- 役割: 単体大技や全体化した攻撃を大きく押し上げる爆発ターン始動札。

### 4. `きゅっとしてドカーン`

- レア度: `レア`
- コスト: `3`
- 種別: `アタック`
- 対象: `単体敵`
- 効果: 敵1体に `28` ダメージ。`狂気` なら全体化する
- アップグレード: ダメージ `36`
- 役割: ボス戦の主砲。`狂気` と `フォーオブアカインド` の受け先。

## Rare Package Notes

- レアは `目を置く` より `置いた盤面をどう勝ちに変えるか` を担当する。
- `フォーオブアカインド` は明確なハイリスク札として、持続型のまま尖らせる。
- `カタストロフビジョン` は目を割る価値を直接盤面火力へ変換する。
- `狂気解放` と `きゅっとしてドカーン` で、ボス単体相手でも勝ち筋が残るようにする。
- レア帯では `狂気` の供給を増やすが、エナジー加速はあくまで初期レリックと撃破テンポに寄せる。

## Madness Implementation Notes

- `狂気` は現在、パワー本体と共通層パッチの両方を実装済み。
- 確認済み:
  - `StrikeIronclad` は `狂気` 中に敵全体対象として解決できる。
  - `Headbutt` は `狂気` 中に共通層パッチだけで敵全体対象として解決できる。
- 共通層パッチで行っていること:
  - アタックの `TargetType.AnyEnemy` / `TargetType.RandomEnemy` を `TargetType.AllEnemies` に変換する。
  - `OnPlayWrapper` で単体前提カードの仮ターゲットを補完する。
  - `AttackCommand.Targeting(...)` を `TargetingAllOpponents(...)` に差し替える。
- 既知メモ:
  - `AttackCommand.Targeting(...)` の Harmony Prefix では、元メソッドをスキップする場合に戻り値の `AttackCommand` を必ず返す必要がある。
  - `StrikeIronclad` と `Headbutt` は、現時点では共通層パッチだけで通る前提に整理している。
- 未確認:
  - ランダム多段攻撃の既存カードで、`狂気` による全体化が期待どおりに解決するか。
  - X コストや特殊ターゲティング系の既存アタックが共通層パッチでどこまで拾えるか。
